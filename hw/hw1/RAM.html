Random access memory (RAM)

Random access memory, or RAM is an extremely fast, sort-term type of computer memory which temporarily stores all the information your computer needs right now and in the near future. However, RAM is what is described in computing as “volatile,” lacking the ability to retain data during power loss or a system crash.
